# 📊 Data Analyzer & Transformer

## 📌 Description

This Python project is a program that performs data analysis and transformation on a list of numbers.

It allows users to input data and perform operations like summary statistics, factorial calculation, filtering, and sorting.

---

## 🚀 Features

* Input and store data
* Display summary (min, max, sum, average)
* Calculate factorial using recursion
* Filter data using lambda function
* Sort data (ascending/descending)
* Display dataset statistics

---

## 🛠️ Tech Used

* Python (Basics)
